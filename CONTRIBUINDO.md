# Como Contribuir

Obrigado pelo interesse em contribuir com este projeto!

## Passos para contribuir

1. Faça um fork do repositório
2. Crie uma branch com a sua feature: `git checkout -b minha-feature`
3. Faça o commit das suas alterações: `git commit -m "Adiciona minha feature"`
4. Envie para o repositório remoto: `git push origin minha-feature`
5. Abra um Pull Request descrevendo as mudanças

## Boas práticas

- Escreva mensagens de commit claras e objetivas
- Mantenha o código legível e bem organizado
- Teste suas alterações antes de abrir o PR
